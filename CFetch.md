# CFetch
Project home page: https://github.com/clieg/cfetch.git

A simple system information tool for Linux.

### Installation
```
cd $HOME
git clone https://github.com/clieg/cfetch.git
cd ~/cfetch
make compile
sudo make install
```

### Uninstallation
```
cd ~/cfetch
sudo make uninstall
rm -rf ~/cfetch
```
